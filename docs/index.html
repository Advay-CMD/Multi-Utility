✨ Packed With Features

Discover the massive array of tools included in one simple script.

System Report & Info

Lists CPU, Battery, Memory, RAM, and GPU info. Stores reports locally for later use.

Privacy Respected: Data is NOT collected. Stored in System32\Multi_Utility\Reports\.

Virus Scanners

Scan individual suspicious files or perform a deeply comprehensive full-system scan against malicious software.

Network Speed Tester

An integrated, highly reliable speed tester. Maintained extensively to beat default web-based testers.

Account Manager

Add or delete system user accounts locally directly from the CLI tool without navigating infinite settings menus.

Disk Utilities

Format disks or attempt repairs on corrupted PenDrives. It mounts your drive post-wipe.

WARNING: Wipes all data! Use only if you know what you're doing.

Search & Hide Files

Blazing fast V1.2 search (faster than Explorer) without background indexing. Hide and unhide files smoothly.

Casual usage only. Not immune against real hackers!

Getting Started

Installing Multi-Utility is as simple as it gets. No heavy installers, no bloatware.

  • 1
    Extract the Archive Unzip the downloaded .tar file onto your computer from the GitHub link.
  • 2
    Run the Installer Double click on InstallerMultiUtility.bat.
  • 3
    Allow Permissions Accept the UAC (User Account Control) prompt by choosing "Yes".
  • 4
    Wait & Exit Wait for data to be written. Press any key to exit. A shortcut will appear on your desktop!

Dependencies / Compiling

# Notes on dependencies:

Ensure WMIC is installed via Windows Programs and Features, as many functions rely on it.


# Source code compiling:

Batch scripts do not require compilation.

For nested C/Python files, use GCC or MINGW64.
